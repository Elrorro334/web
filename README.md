# 💍 Wedding Invitation - Jennifer & Armando 2026

A premium, interactive digital wedding invitation built for **Invitaciones4You**. This project features smooth animations, an RSVP system, a customized Countdown timer, and a dedicated "Good Wishes" modal.

## 🌟 Features
* **Interactive Entrance:** 3D card extraction and elegant split-screen curtain reveal animations using Framer Motion.
* **Countdown Timer:** Accurately synced to the event time (4:00 PM) in Bakersfield, CA (Pacific Time).
* **RSVP & Good Wishes System:** Automated email notifications sent directly to the bride and groom's inboxes via custom API routes.
* **Smart UI/UX:** One-tap hashtag clipboard copy, responsive split-screen designs, custom SVG icons, and an elegant Emerald, Ivory, and Gold color palette.

## 💻 Tech Stack
* **Framework:** This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app)[cite: 1].
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Notifications:** React Hot Toast
* **Email Handling:** Nodemailer (API Routes)
